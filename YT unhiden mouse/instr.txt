Конечно! Вот пример кода, который можно использовать для изменения стилей CSS на странице YouTube и сделать курсор мыши постоянно видимым:

javascript
Copy code
// получаем элемент, на котором нужно изменить стили
const videoPlayer = document.querySelector('.video-stream.html5-main-video');
// добавляем событие на вход курсора в область видеоплеера
videoPlayer.addEventListener('mouseenter', () => {
  // добавляем стили для курсора мыши
  videoPlayer.style.cursor = 'auto';
});
// добавляем событие на выход курсора из области видеоплеера
videoPlayer.addEventListener('mouseleave', () => {
  // возвращаем стандартный стиль курсора мыши
  videoPlayer.style.cursor = '';
});
Этот код добавляет события на вход и выход курсора мыши из области видеоплеера и изменяет стиль курсора в соответствии с этими событиями. После выполнения этого кода курсор мыши будет всегда видимым во время воспроизведения видео.

Обратите внимание, что этот код работает только для видеоплеера на странице YouTube и может не работать на других сайтах. Также, если вы используете какое-либо расширение для YouTube, это может повлиять на работу этого кода.

Вот пошаговая инструкция по созданию простого расширения для Google Chrome:

Создайте новую папку на вашем компьютере и назовите ее, например, "My Extension".

В папке "My Extension" создайте файл с именем "manifest.json". Этот файл будет содержать информацию о вашем расширении.

Откройте файл "manifest.json" в текстовом редакторе и добавьте следующий код:

json
Copy code
{
  "name": "My Extension",
  "version": "1.0",
  "manifest_version": 2,
  "content_scripts": [{
    "matches": ["*://*.youtube.com/*"],
    "js": ["content.js"]
  }]
}
Этот код определяет название, версию и другие параметры вашего расширения, а также указывает, что ваше расширение будет работать на страницах YouTube и будет включать скрипт, хранящийся в файле "content.js".

В папке "My Extension" создайте файл с именем "content.js" и добавьте в него код, который мы использовали ранее для изменения стилей на странице YouTube:

const videoPlayer = document.querySelector('.video-stream.html5-main-video');

Перейдите на страницу YouTube, включите какое-либо видео и убедитесь, что курсор мыши не исчезает во время воспроизведения.

Круто! Как добавить иконку?
Чтобы добавить иконку для вашего расширения, вам нужно внести некоторые изменения в файл "manifest.json".

Вот как вы можете добавить иконку:

Создайте файл с именем "icon.png" или "icon.jpg" в папке "My Extension". Размер иконки должен быть 128x128 пикселей.

В файле "manifest.json" добавьте свойство "icons", указывающее на путь к файлу иконки:

json
Copy code
{
  "name": "My Extension",
  "version": "1.0",
  "manifest_version": 2,
  "content_scripts": [{
    "matches": ["*://*.youtube.com/*"],
    "js": ["content.js"]
  }],
  "icons": {
    "128": "icon.png"
  }
}
Сохраните изменения в файле "manifest.json".

Перезагрузите ваше расширение, чтобы изменения вступили в силу. Для этого перейдите на страницу chrome://extensions/, найдите свое расширение и нажмите на кнопку "Обновить".

Теперь вы должны увидеть иконку для вашего расширения в панели инструментов браузера.
